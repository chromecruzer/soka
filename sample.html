(Node.js) Training Docs

1. Make sure you have the following software installed:

  • Node.js version 20 LTS preferred (https://nodejs.org/en/download)
  • Git (https://git-scm.com/downloads)
  • VSCode (https://code.visualstudio.com/)

2. After successful installation, check their availability:

  • node --version
  • npm --version
  • code --version
  • git --version

3. To initialize a Node.js project:

  • Create a directory.
  • Enter into the directory and type 
    npm init -y
    in the command terminal.
  • Create a file named 
    index.js
    inside the directory.
  • Inside the command terminal, let's install the Express library. Type 
    npm install express
    .
  • To rerun the project using Node.js, install 
    nodemon
    globally. Inside the command terminal, type 
    npm install nodemon --global
    .
  • After installing `nodemon`, check its availability by typing 
    npm list --g --depth 0
    in the terminal.

4. Modify package.json for ES6 Modules:

Since we're using ES6-based code, you need to update your package.json file by adding the following line:

"type": "module"

This enables ES6 module syntax for import/export.

5. To run the project:

In the VSCode terminal, type:

nodemon index.js

You should be able to see the output in the terminal.

6. Code in index.js:


import express from 'express'
const app = express()

// Handlers
app.get('/soka/:value', (req, res)=>{
    let value = req.params
    console.log(req.method)
    console.log(`From Client ${JSON.stringify(value, null, 2)}`)
    let valueToClient = JSON.stringify(value, null, 2)
    res.send(valueToClient)
})

app.listen(6002, ()=>{
    console.log(`server is listening to port 6002`)
})
